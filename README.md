# 📰 Central Mindanao Newswatch (CMN)

A modern and responsive landing page for the **Central Mindanao Newswatch** mobile application.
This website allows users to explore features and download the APK directly.

---

## 🌐 Live Demo

👉 https://asphyxialou.github.io/CentralMindanaoNewswatch/

---

## 📱 Features

* ⚡ Real-time News Updates
* 🧠 Smart Feed (personalized content)
* 🔴 Live Updates
* 🌦 Weather Integration (API-based forecasts)
* 💬 In-App Communication System
* 📥 Direct APK Download

---

## 🎨 UI/UX Highlights

* Clean modern gradient design
* Interactive phone mockup with hover tilt effect
* Smooth scroll animations
* Responsive layout (mobile, tablet, desktop)
* Animated navigation with hover effects

---

## 🛠️ Built With

* HTML5
* CSS3 (Flexbox + Grid + Media Queries)
* JavaScript (Vanilla JS)
